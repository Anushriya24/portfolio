Overview

Can an LLM improve its own prompts without human intervention? This project implements OPRO (Optimisation by PROmpting) — a framework where a language model acts as an optimiser, iteratively generating better task prompts based on previous performance scores.

The result: task accuracy improved from 0.62 → 0.89 over 10 iterations, with zero manual prompt editing.

The OPRO Loop

Split dataset — train (rows 0–19) and held-out test set (rows 20–39)

Baseline — evaluate initial prompt on training set, record score

Meta-prompt — feed score history to the LLM: "here are previous prompts and their scores — generate a better one"

Extract & evaluate — parse the new prompt, score it on the training set

Repeat — 10 iterations, keeping the best-performing prompt

Test — evaluate best prompt on held-out test set to validate generalisation

What is OPRO?

OPRO (Yang et al., 2023 — Google DeepMind) treats prompt optimisation as a black-box problem. Rather than gradient descent, it uses the LLM itself to generate candidate solutions guided by natural language feedback. This makes it applicable to any task where you can score outputs, without needing access to model weights or gradients.
