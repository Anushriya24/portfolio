Overview

London's property market is one of the most complex in the world — a flat in Kensington can cost 10× the equivalent in Barking. This interactive Tableau dashboard makes that complexity navigable, exploring how time, property type, and location jointly drive house prices across London.

Built for non-technical stakeholders: dynamic filters let anyone explore borough-level trends without any data background.

Features

Borough-level choropleth map showing average prices across all London boroughs
Time series view of price trends across multiple years
Property type breakdown — detached, semi-detached, terraced, flats
Dynamic cross-filters — click any borough to filter all charts simultaneously

Key Insights

Inner London boroughs (Kensington & Chelsea, Westminster, Camden) consistently command the highest prices across all property types
Flats dominate volume in central boroughs; detached properties skew heavily outer London
Outer boroughs (Waltham Forest, Lewisham) showed strong relative price growth in recent years
